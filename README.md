#### App Features:
- Tinder Like Swiper
- Dark Theme
- API integration
- Latest and Breaking News

#### Technology used:
- React Native
- Expo SDK
- TypeScript

#### Installation and Running
- Clone the project `git clone git@github.com:mstva/newsviy.git`
- Open in Visual Studio Code
- Run and compile `npm start or yarn start`
